# RasaChatbot

## Installation

1. Clone the repo
2. Move to chatbot folder and run 
`python3 -m venv ./venv`
`source ./venv/bin/activate`
`pip install -r requirements.txt`
3. Test by running `rasa train` and `rasa shell`
